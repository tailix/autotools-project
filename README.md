Usage
-----

Don't forget to change the following files:

* `.gitignore`
* `AUTHORS`
* `configure.ac`
* `COPYING`
* `Makefile.am`
* `README.md`
* `VERSION`

Remove directory `include` and submodule `vendor/cross` if you don't need them.
